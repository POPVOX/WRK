# WRKBench logo — "Bench rule"

The wordmark is typographic: **WRK** set roman + **Bench** set italic in Newsreader
(weights 600 / 400 italic), sitting on the bench rule — seat in ink, two rust legs.

## Files
- wrkbench-lockup-light(.svg / -bg) — ink on transparent / on cream #FAF7F1
- wrkbench-lockup-dark(.svg / -bg) — cream on transparent / on ink #26221C
- bench-rule-ink.svg / bench-rule-cream.svg — the rule alone (doubles as a section divider)
- wrkbench-lockup-light.png / dark.png — raster renders, 2x

## Colors
- Ink #26221C · Cream #F7F3EC · Paper #FAF7F1
- Rust #8A4B2D (on light) · Rust-bright #C97B4A (on dark)

## Type
- Newsreader (Google Fonts). "WRK" 600 roman, "Bench" 400 italic, letter-spacing -0.015em.
- The SVG lockups reference Newsreader by name — install it or convert text to outlines
  in Illustrator/Figma before using where the font may be missing (PNGs are safe anywhere).

## Rules of use
- Rule width matches the wordmark width; legs inset ~7% from each end; seat = ~23% of rule height.
- Gap between baseline and rule ≈ 0.18× cap height.
- Clear space: one "W" width on all sides. Minimum wordmark size 17px tall (sidebar spec).
- Don't recolor the legs except ink→rust swap shown here; don't stretch the rule without the wordmark.
